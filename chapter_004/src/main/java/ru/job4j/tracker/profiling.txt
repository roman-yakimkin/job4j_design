Запустил jconsole
На вкладке VM Summary можно увидеть характеристики по умолчанию - сколько памяти, какой сборщик мусора используется и т.д.
В принципе, в каждой вкладке можно что-то посмотреть полезное.
Запустил jps.
Узнал id данного процесса
Далее посмотрел разные харатеристики с помощью
jstat -опция id
Далее посмотрел статистику по классам с помощью
jmap -clstats id
При попытке выйти за пределы памяти jconsole просто дисконнектилась.
